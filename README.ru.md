# MacMac 🦊

Расширение для Firefox с контейнерным управлением на основе учётных записей.

[![Лицензия MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Создано на WXT](https://img.shields.io/badge/built%20with-WXT-0052cc)](https://wxt.dev)

## Возможности

- **Изоляция учётных записей** — каждый контейнер对应 одной учётной записи на сайте. Сессии, куки и данные разных аккаунтов на одном сайте полностью разделены.
- **Автоматическое переключение контейнеров** — при посещении сайта MacMac сам направляет вас в нужный контейнер, без лишних кликов.
- **Запоминание по сайтам** — последняя использованная учётная запись для каждого сайта сохраняется между сессиями.
- **Всплывающее окно** — управление всеми учётными записями для текущего сайта в одном месте.
- **Открыть без переключения** — открыть другой аккаунт в новой вкладке, не меняя выбранный по умолчанию. Удобно для быстрой проверки второго аккаунта.
- **Полный цикл из popup** — создание, переименование, удаление и открытие аккаунтов во вкладках.

## Как это работает

Откройте popup на любом сайте и нажмите **Создать новую учётную запись**. MacMac создаёт безымянный контейнер (автоматически именуется «Аккаунт 1», «Аккаунт 2» и т.д.), открывает сайт в нём, и вы входите. Позже можно переименовать аккаунт через popup.

Внутри каждый контейнер называется `ИмяАккаунта (имя_хоста)` — например, `Аккаунт 1 (facebook.com)` — это привязывает контейнер к сайту.

## Реализация

За всё отвечают три ключа хранилища:

- `accounts` — метаданные аккаунтов (id, имя, хосты)
- `hostnameAccounts` — соответствие хоста и аккаунтов
- `lastSelected` — предпочтительный аккаунт для каждого хоста

При переходе на сайт фоновый скрипт проверяет `lastSelected`. Если аккаунт назначен, вкладка переключается на него. Если нет — используется личность по умолчанию (без контейнера).

## Использование

**Создание аккаунта:** нажмите на значок на панели инструментов, затем на **+ Создать новую учётную запись**. Создаётся новый безымянный контейнер и активируется. Войдите в свой аккаунт, при желании переименуйте его через popup.

**Переключение аккаунтов:** нажмите на карточку любого аккаунта в popup. Ваш выбор сохраняется для будущих визитов.

**Открытие без переключения:** нажмите на значок новой вкладки на карточке аккаунта, чтобы открыть сайт в этом контейнере, не меняя привязку по умолчанию.

## Разработка

```bash
bun install           # установка зависимостей + WXT postinstall
bun run dev           # dev-сервер с горячей перезагрузкой
bun run compile       # проверка типов
bun run lint          # линтер
bun run build:firefox # сборка для продакшена
```
