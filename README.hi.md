# MacMac 🦊

Firefox के लिए अकाउंट-आधारित कंटेनर प्रबंधन।

[![MIT लाइसेंस](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![WXT से बना](https://img.shields.io/badge/built%20with-WXT-0052cc)](https://wxt.dev)

## सुविधाएँ

- **अकाउंट आइसोलेशन** — हर कंटेनर एक वेबसाइट पर एक अकाउंट का प्रतिनिधित्व करता है। एक ही साइट पर अलग-अलग अकाउंट्स की सेशन, कुकीज़ और डेटा पूरी तरह अलग रहते हैं।
- **ऑटो कंटेनर स्विचिंग** — साइट पर जाते ही MacMac आपको सही कंटेनर में भेज देता है, बिना किसी क्लिक के।
- **प्रति साइट मेमोरी** — हर साइट के लिए आखिरी इस्तेमाल किया गया अकाउंट सेशन के बीच में भी याद रहता है।
- **पॉपअप इंटरफेस** — मौजूदा साइट के सभी अकाउंट्स एक जगह से मैनेज करें।
- **बिना स्विच किए खोलें** — डिफ़ॉल्ट बदले बिना नए टैब में दूसरा अकाउंट खोलें। दूसरा अकाउंट जल्दी चेक करने के लिए काम का है।
- **पॉपअप से पूरा कंट्रोल** — अकाउंट बनाएँ, नाम बदलें, मिटाएँ, और नए टैब में खोलें।

## कैसे काम करता है

किसी भी वेबसाइट पर पॉपअप खोलें और **नया अकाउंट बनाएँ** पर क्लिक करें। MacMac एक बिना नाम का कंटेनर बनाता है (अपने आप "अकाउंट 1", "अकाउंट 2" वगैरह नाम मिलता है), उसमें साइट खोलता है, और आप लॉग इन करते हैं। बाद में पॉपअप से नाम बदल सकते हैं।

हर कंटेनर का अंदरूनी नाम `अकाउंट_का_नाम (होस्टनेम)` होता है — जैसे `अकाउंट 1 (facebook.com)` — जो इसे उस साइट से जोड़ता है।

## तकनीकी जानकारी

तीन स्टोरेज की सब कुछ संभालती हैं:
- `accounts` — अकाउंट मेटाडेटा (id, नाम, होस्टनेम)
- `hostnameAccounts` — होस्टनेम से अकाउंट का मैपिंग
- `lastSelected` — हर होस्टनेम के लिए पसंदीदा अकाउंट

जब आप किसी साइट पर जाते हैं, बैकग्राउंड स्क्रिप्ट `lastSelected` चेक करता है। अगर कोई अकाउंट मैप है, तो टैब उस कंटेनर में स्विच हो जाता है। नहीं तो डिफ़ॉल्ट पहचान (बिना कंटेनर) इस्तेमाल होती है।

## इस्तेमाल

**अकाउंट बनाना:** टूलबार आइकन पर क्लिक करें, फिर **+ नया अकाउंट बनाएँ** पर। एक नया बिना नाम का कंटेनर बनता है और एक्टिवेट होता है। अपने अकाउंट में लॉग इन करें, चाहें तो पॉपअप से नाम बदल लें।

**अकाउंट स्विच करना:** पॉपअप में किसी भी अकाउंट कार्ड पर क्लिक करें। आपकी पसंद अगली विज़िट के लिए सेव होती है।

**बिना स्विच किए खोलना:** अकाउंट कार्ड पर नए टैब के आइकन पर क्लिक करें, ताकि उस कंटेनर में साइट खुले बिना डिफ़ॉल्ट बदले।

## डेवलपमेंट

```bash
bun install           # डिपेंडेंसी इंस्टॉल + WXT postinstall
bun run dev           # डेव सर्वर हॉट रीलोड के साथ
bun run compile       # टाइप चेक
bun run lint          # लिंट
bun run build:firefox # प्रोडक्शन बिल्ड
```
