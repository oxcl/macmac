How It Works

See how MacMac transforms the way you manage accounts in Firefox.

01

The Problem

Juggling multiple accounts on the same website is frustrating. Constant log in and log out, shared sessions that mix up your data, and browser profiles that don't talk to each other. It's a productivity killer — and a privacy risk.

02

Create Accounts

With MacMac, create separate accounts for each of your identities. Work, Personal, School — all on the same website. Each account is backed by a Firefox Container, keeping cookies, sessions, and data completely isolated from one another.

03

Automatic Switching

MacMac remembers which account you used last for each website. When you navigate to a site, it automatically switches to the right container in the background. No more manual picking — just browse and MacMac handles the rest.

04

Seamless Workflow

Every website opens in the right container automatically. Your accounts stay organized, your sessions stay separate, and your workflow stays smooth. No more context switching — just a clean, productive browsing experience.

Frequently Asked Questions

How is MacMac different from Firefox's built-in Container feature?

Firefox's built-in Container feature lets you manually assign tabs to containers. MacMac builds on that by automatically remembering which container you used last for each website and switching to it — no manual selection required. It also provides a dedicated popup interface for managing all your accounts in one place.

Is my data private? Does MacMac collect any information?

MacMac is fully private. All data — your container mappings, preferences, and settings — is stored locally in your browser's storage using the WebExtensions Storage API. MacMac does not collect, transmit, or share any personal information, browsing history, or usage data. The extension is open source, so you can verify this yourself.

Can I use MacMac with websites I'm already logged into?

Yes. When you create a new container for a website, it starts with a clean slate — separate cookies, storage, and sessions. You can log into your existing accounts from within the new container without affecting your other sessions. Each container is fully isolated from the others.

Which Firefox versions are supported?

MacMac requires Firefox 89 or later, as it relies on the contextualIdentities API which is available in these versions. The extension is tested against the latest stable release of Firefox and Firefox Developer Edition.

How do I uninstall MacMac?

Go to Firefox's Add-ons and Themes page (about:addons), find MacMac in the Extensions list, and click Remove. Your containers and their data will remain in Firefox, but MacMac will no longer manage automatic switching for them.

Is MacMac open source?

Yes. MacMac is completely open source. You can view the source code, report issues, and contribute on GitHub. The extension is free to use and always will be.
