Privacy Policy

Introduction

MacMac is an open-source Firefox extension for multi-account container management. The source code is publicly available on GitHub under the MIT License. This Privacy Policy explains what data MacMac does and does not collect.

No Data Collection

MacMac does not collect, transmit, or share any personal information, browsing history, or usage data. The extension operates entirely within your browser and has no external servers to send data to.

Local Storage

All data created by MacMac is stored exclusively in your browser's local storage using the WebExtensions Storage API. This includes:

  • Container profile names and metadata
  • Hostname-to-container mappings
  • Last-selected container preferences

This data never leaves your browser. You can inspect it at any time via about:debugging in Firefox and clear it by removing the extension.

No Third-Party Services

MacMac does not integrate any third-party analytics, telemetry, tracking, advertising, or external services. There are no cookies set by the extension, no network requests made, and no remote code loaded.
