# MacMac 🦊

إضافة Firefox لإدارة الحاويات بناءً على الحسابات.

[![رخصة MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![مبني بـ WXT](https://img.shields.io/badge/built%20with-WXT-0052cc)](https://wxt.dev)

## المميزات

- **عزل الحسابات** — كل حاوية تمثل حساباً واحداً على الموقع. الجلسات والكوكيز والبيانات تبقى منفصلة تماماً بين الحسابات على نفس الموقع.
- **التبديل التلقائي للحاويات** — عند زيارة موقع، MacMac يوجهك للحاوية الصحيحة بدون أي نقرات.
- **حفظ لكل موقع** — آخر حساب استخدمته لكل موقع يُحفظ بين الجلسات.
- **نافذة منبثقة** — إدارة جميع حساباتك للموقع الحالي في مكان واحد.
- **فتح بدون تبديل** — فتح حساب آخر في لسان جديد دون تغيير الحساب الافتراضي. مفيد لمراجعة حساب ثانٍ بسرعة.
- **إدارة كاملة من النافذة** — إنشاء وإعادة تسمية وحذف وفتح الحسابات في ألسنة جديدة.

## كيف يعمل

افتح النافذة المنبثقة على أي موقع وانقر **إنشاء حساب جديد**. MacMac ينشئ حاوية بدون اسم (تُسمى تلقائياً "حساب 1"، "حساب 2"، إلخ)، يفتح الموقع فيها، وأنت تسجل الدخول. يمكنك إعادة تسمية الحساب لاحقاً من النافذة.

كل حاوية تُسمى داخلياً `اسم_الحساب (المضيف)` — مثلاً `حساب 1 (facebook.com)` — وهذا يربطها بالموقع.

## التفاصيل التقنية

ثلاثة مفاتيح تخزين تتعامل مع كل شيء:

- `accounts` — بيانات الحسابات (المعرف، الاسم، المضيفات)
- `hostnameAccounts` — ربط المضيف بالحسابات
- `lastSelected` — الحساب المفضل لكل مضيف

عند الانتقال لموقع، يتحقق السكريبت في الخلفية من `lastSelected`. إذا كان هناك حساب مرتبط، ينتقل اللسان إليه. إذا لا، تُستخدم الهوية الافتراضية (بدون حاوية).

## الاستخدام

**إنشاء حساب:** انقر على أيقونة شريط الأدوات، ثم **+ إنشاء حساب جديد**. تُنشأ حاوية جديدة بدون اسم ويتم تفعيلها. سجل الدخول لحسابك، وأعد تسميته من النافذة إن أردت.

**التبديل بين الحسابات:** انقر على أي بطاقة حساب في النافذة. اختيارك يُحفظ للزيارات القادمة.

**فتح بدون تبديل:** انقر على أيقونة اللسان الجديد على بطاقة الحساب لفتح الموقع في تلك الحاوية دون تغيير الربط الافتراضي.

## التطوير

```bash
bun install           # تثبيت التبعيات + WXT postinstall
bun run dev           # خادم التطوير مع إعادة تحميل فورية
bun run compile       # فحص الأنواع
bun run lint          # فحص الكود
bun run build:firefox # بناء الإنتاج
```
