BoxBox Firefox 多账号容器管理器

多个账号。同一网站。完全隔离。

一款 Firefox 扩展，让你在同一网站上创建和切换多个账号。每个账号运行在独立容器中，拥有独立的 cookies、会话和数据。

在 Firefox 附加组件中获取 从 GitHub 下载

工作原理

BoxBox 如何管理同一网站上的多个账号。

01

问题所在

在同一网站上管理多个账号令人头疼。你得反复登录登出，丢失页面进度，还可能不小心用错账号发帖。浏览器配置文件能帮上忙，但它们很重，而且无法共享书签或扩展。

02

创建账号

为同一网站创建多个账号。每个账号由独立的 Firefox 容器支持，cookies、会话和数据完全互相隔离。

03

自动切换

BoxBox 会记住你在每个网站上最后使用的账号。当你访问某个网站时，它会自动切换到对应的容器。你只需正常浏览。

04

开箱即用

设置完成后，每个网站都会在正确的容器中打开。你的账号保持分离，会话保持干净，无需操心。

常见问题

BoxBox 和 Firefox 内置的容器功能有什么区别？

Firefox 内置的容器功能需要你手动将标签页分配到容器。BoxBox 在此基础上自动记住你在每个网站上最后使用的容器并切换过去——无需手动选择。它还提供了一个专门的弹窗界面，让你在一个地方管理所有账号。

我的数据安全吗？BoxBox 会收集信息吗？

BoxBox 完全保护隐私。所有数据——你的容器映射、偏好设置和配置——都通过 WebExtensions Storage API 存储在浏览器本地。BoxBox 不会收集、传输或分享任何个人信息、浏览历史或使用数据。该扩展是开源的，你可以自行验证。

我可以在已经登录的网站上使用 BoxBox 吗？

可以。当你为某个网站创建新容器时，它从一张白纸开始——独立的 cookies、存储和会话。你可以在新容器中登录已有账号，不会影响其他会话。每个容器与其他容器完全隔离。

支持哪些 Firefox 版本？

BoxBox 需要 Firefox 89 或更高版本，因为它依赖于这些版本中提供的 contextualIdentities API。该扩展针对最新的 Firefox 稳定版和 Firefox Developer Edition 进行测试。

如何卸载 BoxBox？

前往 Firefox 的附加组件和主题页面（about:addons），在扩展列表中找到 BoxBox，点击移除。你的容器及其数据将保留在 Firefox 中，但 BoxBox 将不再管理它们的自动切换。

BoxBox 是开源的吗？

是的。BoxBox 完全开源。你可以在 GitHub 上查看源代码、报告问题和参与贡献。该扩展免费使用，并将永远免费。

BoxBox 与 Mozilla 的 Multi-Account Containers 扩展兼容吗？

不兼容。BoxBox 和 Mozilla 的 Multi-Account Containers 采用了两种不同的容器管理方式，彼此不兼容。同时运行两个扩展可能会导致冲突和意外行为。
