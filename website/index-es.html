BoxBox Gestor de cuentas múltiples para Firefox

Varias cuentas. Un mismo sitio. Aislamiento total.

Una extensión de Firefox que te permite crear y cambiar entre múltiples cuentas en un mismo sitio web. Cada cuenta corre en su propio contenedor con cookies, sesiones y datos separados.

Obtener en Firefox Add-ons Descargar desde GitHub

Cómo funciona

Cómo BoxBox gestiona múltiples cuentas por sitio web.

01

El problema

Manejar múltiples cuentas en un mismo sitio web es frustrante. Cierras sesión, vuelves a iniciar, pierdes tu lugar, publicas accidentalmente desde la cuenta incorrecta. Los perfiles del navegador ayudan pero son pesados y no comparten marcadores ni extensiones.

02

Crear cuentas

Crea múltiples cuentas para un mismo sitio web. Cada una respaldada por su propio contenedor de Firefox, manteniendo cookies, sesiones y datos completamente aislados entre sí.

03

Cambio automático

BoxBox recuerda qué cuenta usaste por última vez en cada sitio web. Cuando navegas a un sitio, cambia al contenedor correcto automáticamente. Solo navega.

04

Funciona y ya

Una vez configurado, cada sitio web se abre en el contenedor correcto. Tus cuentas permanecen separadas, tus sesiones limpias, y no tienes que pensar en ello.

Preguntas frecuentes

¿En qué se diferencia BoxBox de la función de contenedores integrada en Firefox?

La función de contenedores integrada en Firefox te permite asignar manualmente pestañas a contenedores. BoxBox se basa en eso recordando automáticamente qué contenedor usaste por última vez en cada sitio web y cambiando a él — sin selección manual. También proporciona una interfaz popup dedicada para gestionar todas tus cuentas en un solo lugar.

¿Mis datos son privados? ¿BoxBox recopila información?

BoxBox es completamente privado. Todos los datos — tus mapeos de contenedores, preferencias y configuración — se almacenan localmente en el almacenamiento de tu navegador usando la API WebExtensions Storage. BoxBox no recopila, transmite ni comparte ninguna información personal, historial de navegación ni datos de uso. La extensión es de código abierto, así que puedes verificarlo tú mismo.

¿Puedo usar BoxBox en sitios web donde ya tengo sesión iniciada?

Sí. Cuando creas un nuevo contenedor para un sitio web, comienza desde cero — cookies, almacenamiento y sesiones separados. Puedes iniciar sesión en tus cuentas existentes desde el nuevo contenedor sin afectar tus otras sesiones. Cada contenedor está completamente aislado de los demás.

¿Qué versiones de Firefox son compatibles?

BoxBox requiere Firefox 89 o posterior, ya que depende de la API contextualIdentities disponible en estas versiones. La extensión se prueba con la última versión estable de Firefox y Firefox Developer Edition.

¿Cómo desinstalo BoxBox?

Ve a la página de Complementos y Temas de Firefox (about:addons), encuentra BoxBox en la lista de Extensiones y haz clic en Eliminar. Tus contenedores y sus datos permanecerán en Firefox, pero BoxBox ya no gestionará el cambio automático por ellos.

¿BoxBox es código abierto?

Sí. BoxBox es completamente de código abierto. Puedes ver el código fuente, reportar problemas y contribuir en GitHub. La extensión es gratuita y siempre lo será.

¿Es BoxBox compatible con la extensión Multi-Account Containers de Mozilla?

No. BoxBox y Multi-Account Containers de Mozilla adoptan enfoques diferentes para la gestión de contenedores y no son compatibles. Usar ambas extensiones a la vez puede causar conflictos y comportamientos inesperados.
